Features
Everything field teams actually need.
📁

Smart Folder Hierarchy

Project → Location → Category → Sub. Photos auto-inherit GPS, timestamps, and folder metadata.

📋

Reusable Templates

Create a template once with folder structure, camera settings, and required shots. Clone for every new project.

📸

Quick Capture

Open folder → tap → done. Auto-watermark with date, GPS, and project name. Burst mode for rapid documentation.

🗺️

Map View

Every photo pinned on a map. See all capture points per project. Geofence auto-suggests the right project.

📤

Export & Reports

PDF reports with photos + metadata + map. ZIP exports preserving folder structure. One-tap sharing.

✈️

Offline First

100% local. No account, no cloud required. Works in dead zones. Sync when you're ready.
